Loan Approval System




Introduction

• Indian Financial sector has seen an tremendous growth in last decade, but this growth is not without any glitches.
• According to PWC and Deloitte, there is a dramatic increase in volume of bank loan frauds. It is the major concern
• Hence, techniques are necessary for early prediction of loan being non performing asset (NPA) or fraudulent.
• The data which is needed for loan analysis has to be built by considering features like loan amount, purpose, repayment pattern, employment etc.
• This data may not be structured, and needs continuous analysis
• Hence Big Data technology must be used for such analysis, to assist in efficient growth of The Indian Financial sector.








Objective of The Project

• The proposed system is aimed at reducing risks of losses to banks due to bad loans.
• It is based on the analysis of various attributes extracted from the previous loan records.
• The system would make use of an ensemble of machine learning algorithms
• Deploy the model for use with new customer data in a reliable manner
• It will help improve the financial health of the banking sector
• The project will help automate the loan approval system and thus processing loans in a more transparent manner
